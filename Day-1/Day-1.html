HTML

HTML Stands for Hyper Text Markup Language.

HTML is stand markup language for creating website

Heading tag

HTML headings are titles or subtitles that you want to display on a webpage.
HTML headings are defined with the h1 to h6 tags.

Paragraph tag

A Paragraph always start on new line, and is usually a block of text.

Pre tag 
          The pre tag defines preformatted text.

          Text in a pre element is displayed in a fixed-width font,

          and the text preserves both spaces and line breaks.
